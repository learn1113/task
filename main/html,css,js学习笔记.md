# html  
## 基本标签  
### 标题标签  
用`<h1><\h1>`表示，可添加align属性，属性值有left，center和right。规定标题的位置  
### 段落标签  
用`<p><\p>`表示，文本内容放在其中  
### 换行  
`<br>`  
### 水平线  
`<hr>`有`color` `width` `align` `size`属性  
### 图片  
`<img src alt title width hight>` `src`放图片地址 `alt`放无法显示时的文本`title`放鼠标悬停时的提示  
### 超链接  
`<a><\a>`有`href`属性，存放链接地址  
### 文本标签  
`<em><\em>`加重斜体  
`<i><\i>`斜体  
`<b><\b>`加粗  
`<strong><\strong>`加重加粗  
`<del><\del>`删除  
`<span><\span>`无特定含义  
### 列表  
`<ol><li><\li><\ol>`有序列表，`<ol>`有`type`属性，规定序号是什么。可嵌套  
`<ol><li><\li><\ol>`无序列表，与有序列表相似  
### 列表   
快捷键   `table>tr*数字>td*数字{内容}`

`<table><\table>`内有`<td><\td>`表示列，`<tr><\tr>`表示行
